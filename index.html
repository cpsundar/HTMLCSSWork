Summary Lattitude vs X

The purpose of this project is to analyze how weather changes as you get closer to the equator. To accomplish this analysis, we first pulled data from the OpenWeatherMap API to assemble a dataset on over 500 cities randomly.

After assembling the dataset, we used Matplotlib to plot various aspects of the weather vs 1atitule. Factors we looked at included: temperature, humidity, cloudiness and wind speed. This site provides the source and visualizations created as a part of the analysis as well as explanations.
